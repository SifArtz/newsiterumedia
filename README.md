# newsiterumedia

Легкий интерфейс для модерации треков с rumedia.io. Открывайте `index.html` локально или на хостинге — страница подтягивает данные из текущей очереди премиумов или синглов, оставляя только необходимые поля: обложку, исполнителя, название, жанр, дату загрузки, действие и встроенный проигрыватель.

## Как пользоваться
1. Откройте `index.html` в браузере.
2. Вставьте ваш PHPSESSID из rumedia в поле «PHPSESSID для доступа в админку» и нажмите «Сохранить» — значение сохраняется в `localStorage` и добавляется к запросам, включая прокси.
3. Используйте переключатели «Очередь премиумов» и «Очередь синглов», чтобы загрузить нужный список.
4. Кнопка «Обновить» повторно запрашивает активный источник.

> Примечание: данные берутся из HTML-страниц панели (`/admin-cp/manage-songs`) и могут требовать авторизации или разрешённого CORS; при блокировке браузером запросы автоматически выполняются через публичный прокси allorigins.win (с добавленным PHPSESSID, если задан).
