# RUMEDIA Moderation Desk

Небольшая витрина для модераторов: забирает таблицу загруженных треков (премиум и синглы), вытаскивает обложку, исполнителя, название, жанр, дату загрузки, действие и плеер, и показывает в компактной сетке карточек.

## Запуск
Откройте `index.html` в любом веб-сервере (например, `npx serve .` или встроенный hosting). Стили и логика лежат в `/assets`.

## Работа с сессией и прокси
1. Введите `PHPSESSID` из админки в поле «PHPSESSID для доступа в админку» и нажмите «Сохранить». Значение сохранится в `localStorage`.
2. Фронтенд сначала попробует забрать HTML через `proxy.php` (с подстановкой вашего `PHPSESSID` в заголовок `Cookie`), потом прямой запрос, затем резервный `api.allorigins.win`.
3. При желании можно задать целевой URL напрямую: `proxy.php?url=https://rumedia.io/media/admin-cp/manage-songs?check_pro=1&phpsessid=...` или использовать псевдонимы `source=premium|singles` вместо `url`.
4. Пример рабочего PHP-прокси уже в репозитории (`proxy.php`). Разверните его на том же домене, что и фронтенд (чтобы не упереться в CORS), либо адаптируйте под свой сервер. Прокси разрешает только запросы на `https://rumedia.io/media/admin-cp/manage-songs*`.

## Интерфейс
- Кнопки «Очередь премиумов» и «Очередь синглов» переключают источник.
- «Обновить» перезапускает загрузку текущей очереди.
- Тосты показывают статус загрузки/ошибок, карточки содержат бейдж источника, жанр, время загрузки и кнопку «Действие» плюс аудиоплеер.
